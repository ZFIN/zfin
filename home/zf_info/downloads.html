ZFIN Data Reports

 Warranty Disclaimer and Copyright Notice
ZFIN data reports are updated every Sunday. The format for each file is described below. Each record contains a ZFIN ID which may be used to link data between files. All files use tab as the field delimiter and are ordered by ZFIN ID. The 'download' links will display the file on your browser. Use your browser 'save as' feature to make a local copy of the file. If you choose to link to a ZFIN data page from your web site, append the ZFIN ID to the URL http://zfin.org/cgi-bin/ZFIN_jump?record=

Please direct questions or requests for additional data to ZFIN.
