ZFIN Site News & Status

Last updated 2001/01/30

This page contains news about the Zebrafish Information Network web site (zfin.org). This includes scheduled down time and site upgrades. If you have any questions or comments, please contact Pat Edwards.

Main site scheduled to be down on 2001/02/10

2001/01/29: The main site (zfin.org) is tentatively scheduled to be down on 2001/02/10 for maintenance. It is expected to be down for less than 2 hours. During this time the mirror sites in Melbourne,Australia, Mishima, Japan, and Strasbourg, France will be up.

This announcement will become more definitive as the date nears.
