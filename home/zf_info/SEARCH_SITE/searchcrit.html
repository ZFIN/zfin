The Zebrafish Search Machine

This engine will search all HTML files currently available on the ZFIN website for the specified string. Hot links to files containing the search string are returned, along with the sentence containing the search string. This engine searches only for text contained in the "static" pages. Data stored in the relational ZFIN database must be searched from zfin.org/ZFIN/.
