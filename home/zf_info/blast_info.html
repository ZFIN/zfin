BLAST Programs

WU BLAST (Washington University BLAST), developed by Warren Gish, is an implementation of the BLAST (Basic Local Alignment Search Tool) algorithm. The five programs in the WU BLAST version 2.0 are available at ZFIN BLAST.
  • BLASTN - compares a nucleotide query sequence against a nucleotide sequence database.

  • BLASTP - compares an amino acid query sequence against a protein sequence database.

  • BLASTX - compares the six-frame conceptual translation products of a nucleotide query sequence (both strands) against a protein sequence database.

  • TBLASTN - compares an amino acid query sequence against the six-frame translations of a nucleotide sequence database.

  • TBLASTX - compares the six-frame translations of a nucleotide query sequence against the six-frame translations of a nucleotide sequence database.

Nucleotide Sequence Database
  • GenBank Zebrafish - Subset of GenBank that contains zebrafish sequences from these divisions: vetebrate and high-throughput cDNAs (HTC). ESTs, genome survey sequences (GSS), and high-throughput genomic sequences (HTGs) are not included. Zebrafish sequences are parsed from these divisions by examining the organism lines for entries "Danio rerio".

  • RefSeq Zebrafish mRNA - NCBI RefSeq Zebrafish transcripts (Zebrafish RefSeqs of type: NM_, NR, XM_).

  • ZFIN GenBank Sequences - Subset of GenBank that contains zebrafish sequences from all divisions (including EST) that are associated with ZFIN zebrafish genes or markers or clones.

  • ZFIN cDNA Sequences - Combination of GenBank zebrafish cDNA sequences from all divisions (including EST) that are associated with ZFIN zebrafish genes or markers or clones, and Vega zebrafish transcripts that are associated with ZFIN zebrafish genes.

  • ZFIN Genes with Expression - Subset of ZFIN cDNA Sequences for which ZFIN has obtained related gene expression data.

  • ZFIN Morpholino Sequences - Morpholino sequences in ZFIN.

  • ZFIN MicroRNA Sequences - MicroRNA sequences in ZFIN.

  • ZFIN Vega Transcripts - Vega zebrafish transcripts that are associated with ZFIN zebrafish genes.

  • Zebrafish mRNA - Subset of GenBank that contains zebrafish sequences from all divisions (including EST) that are of type mRNA. Zebrafish mRNA sequences are parsed by examining the organism lines for entries "Danio rerio" and locus line for entries containing "mRNA".

  • EST Zebrafish - Subset of GenBank that contains zebrafish sequences from EST division. Zebrafish sequences are parsed from the EST divisions by examining the organism lines for entries "Danio rerio".

  • Ensembl Zebrafish Transcripts - Ensembl zebrafish transcripts.

  • TIGR Zebrafish Clusters - Tentative consensus transcript sequences from TIGR's Zebrafish Gene Index (ZGI).

  • Zebrafish DNA - Subset of GenBank that contains zebrafish sequences from all divisions (including EST) that are of type DNA. Zebrafish DNA sequences are parsed by examining the organism lines for entries "Danio rerio" and locus line for entries containing "DNA".

  • HTG Zebrafish - Subset of GenBank that contains zebrafish High-throughput Genomic sequences (HTG). Zebrafish sequences are parsed from the HTG division by examining the organism line for entries "Danio rerio".

  • GSS Zebrafish - Subset of GenBank that contains zebrafish Genome Survey Sequences (GSS). Zebrafish sequences are parsed from the GSS division by examining the organism line for entries "Danio rerio".

  • Zebrafish Trace Archive - Zebrafish sequences from NCBI Trace Archive database.
Protein Sequence Database
  • RefSeq Zebrafish Protein - NCBI RefSeq zebrafish protein sequences. (Zebrafish sequences of type: NP_, XP_).

  • SwissProt/TrEMBL Zebrafish - Zebrafish protein sequences from the "non-redundant" set of sequences from SwissProt and TrEMBL. Zebrafish sequences are defined as those entries "Danio rerio" in the species lines.
