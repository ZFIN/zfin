Job Opening

Java Architect
The ZFIN Java architect works with a team of biologists and computer scientists to design, develop and maintain a dynamic bioinformatics database infrastructure for integration, consolidation and interpretation of zebrafish gene structures, expression patterns and mutant phenotypes. Applicants should have experience with Java architecture and programming, demonstrated team leadership in a software development environment, SQL, UNIX, and version control software. Experience with multiple relational database engines, physical and logical database design is preferred. Minimum qualifications: Bachelor's degree with 5+ years experience in database administration and Java architecture, Master's preferred. Preference will be given to candidates with knowledge of biology and/or experience connecting web- sites with databases. Applications received by August 29th, 2005, receive first consideration.

More information is available on ZFIN about the staff and the database project.
