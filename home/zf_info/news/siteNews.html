ZFIN Site News & Status

Last updated 2001/05/10

This page contains news about the Zebrafish Information Network web site (zfin.org). This includes scheduled down time and site upgrades. If you have any questions or comments, please contact Pat Edwards.

ZFIN support for Internet Explorer is coming

2001/05/10: A Beta test site, http://test.zfin.org is now available that provides access to ZFIN using Internet Explorer. Version 5.0 or higher is required on either Macintosh and PC platforms. The data at this test site reflects ZFIN data as of May 5, 2001. Changes made on this site will not appear on the main ZFIN site. Please feel free to try this site. It will be available through 2001/05/15. Comments or questions should be directed to Judy Sprague.
