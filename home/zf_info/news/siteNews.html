ZFIN Site News & Status

This page contains news about the Zebrafish Information Network web site (zfin.org). This includes scheduled down time and site upgrades. If you have any questions or comments, please contact Pat Edwards.

Merged Map (ZMAP) Available

2001/06/22: The option of viewing several maps 'merged' onto one backbone is now available.

Support for Internet Explorer added to ZFIN

2001/05/16: You can now access ZFIN using Internet Explorer. Version 5.0 or higher is required on either Macintosh and PC platforms.
