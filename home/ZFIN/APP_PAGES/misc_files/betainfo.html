Current Status of Mutation Data

ZFIN maintains information about mutations including the phenotype, map location, mutagen, parental lineage, and current source for most lines, although the information on some mutants is still incomplete. Many mutant records also include one or more high-resolution images illustrating the phenotype at various developmental stages and/or orientations.

Details:

Number of mutations: about 2000

Number of images: 2043

Coverage: We have attempted to list all known zebrafish mutations, including those produced in screens at Oregon, MGH, and Tuebingen. Some mutations may still be missing, either because we have been unable to collect information on them, or because the information has not yet been made publically available.

Completeness: The completeness of information varies from mutation to mutation, ranging from nothing but allele designation to more or less complete records including one or more images. For most mutations, there is at least a description of the phenotype and the mutation’s origins.

The images associated with mutations are only sparsely described. We plan to classify images by the anatomical structures shown as well as orientation, age, specimen type and so on; for most images, only developmental stage is listed at this time.

Accuracy: We have tried to ensure the accuracy of ZFIN data. Nevertheless, there are errors. Please contact us if you find errors. Ultimately, the completeness and quality of the data in ZFIN depend on the commitment of the research community.

One of the most powerful features of the mutation search interface is that it allows keyword search using a predefined set of defects. The interface that implements this feature is a Java applet that allows you to browse and select keywords from the predefined list. We have found that, although the applet seems to work fine on all tested platforms, some platforms (notably Netscape 3 on Macintosh) may require you to quit Netscape, increase its memory allocation, and then restart Netscape. You will know this problem has occurred when the applet appears, but contains no list of keywords to select. Also, note that the applet may take a few moments to appear, particularly on slower computers.

Please report erroneous data, interface bugs, and other problems by sending email to zfinadmn@zfin.org. A convenient shortcut is the "questions and comments" button at the top left of every ZFIN screen, which automatically starts an email to the design team.

ZFIN has been established as the official nomenclature clearing house for the zebrafish research community. Automated registration of new nomenclature (e.g. locus names, abbreviations, allele designations), as well as regularly updated nomenclature conventions are available. Please direct nomenclature questions to the ZFIN data curator.
