# This is the list of organism for the ZECO taxonomy ontology, a slim of the full NCBI taxonomy ontology
#
# To create a new slim:
# 1. Add additional organism to the list of leaf terms
# 2. Add new root organism to list of root terms if needed:
#    For now we have bacteria, viruses and fungi
# 3. Go to ontofox web site: http://ontofox.hegroup.org/
# 4. Select the NCBITaxon ontology
# 5. Paste the list of root taxa into the ' Include low level source term URIs' field
# 6. Paste the list of leaf taxa into the '  Include top level source term URIs and target direct superclass URIs' field
# 7. select 'include NoIntermediates'
# 8. paste 'includeAllAxioms' into the ' Annotation/Axiom Specification: Include source annotation URIs' field
# 9. Hit 'Get OWL Output File'
